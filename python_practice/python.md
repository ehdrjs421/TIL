## 입출력

### 입력

- `input()` 을 통하여 사용자에게 입력을 요청
- 입력 받은 자료형은 항상 str이므로 필요시 형변환 진행

### 출력

- `print()` 을 통하여 터미널 상에 결과를 출력
- `,` 를 통해 여러 변수 출력 가능
- `sep,end` 를 사용하여 문장의 형태를 변형 가능
